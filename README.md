# Maximum Subarray Sum Project

## 📌 Project Overview

This project focuses on solving the **Maximum Subarray Sum problem**, where the goal is to find the contiguous subarray within a given integer array that has the largest sum.

We implemented and analyzed **two approaches**:

* A **non-recursive algorithm (Kadane’s Algorithm)**
* A **recursive algorithm (Divide and Conquer)**

Each part of the project is divided among team members to cover pseudo code, implementation, analysis, and comparison.
